Услуги / Вариант №1 (основной).
«Интернет-анкетирование»

Согласовываем содержание анкеты, отчетность по результатам исследования, в том числе, при необходимости, Аналитическую карту интерпретируемых событий, и все необходимые нюансы исследования.

Со своей стороны согласно прайс-листу:

  • Формирую электронную анкету проекта, доступную для заполнения по гиперссылке.
  • При необходимости, настраиваю параметры рассылки опрашиваемым электронные письма-приглашения к прохождению анкетирования.
  • При необходимости, настраиваю параметры отбраковки анкет проекта.
  • Вношу переданные мне Вашим представителем сведения об опрашиваемых, подлежащих опросу в Базу опрашиваемых проекта (при необходимости и не предоставлении Базы опрашиваемых мне Вашим представителем в необходимом формате).
  • При необходимости, произвожу выездные очные консультации, в том числе возможно в форме участия в опросах части респондентов.

Также, при необходимости, по договорной цене совместными с Вашим представителем усилиями осуществляю:

  • Работы по организации автоматического размещения гиперссылок на прохождение анкетирований и других необходимых материаллов в личных кабинетах опрашиваемых на Ваших информационных ресурсах.
  • Работы по организации автоматического размещения гиперссылок на прохождение анкетирований и других необходимых материалов в информационных ресурсах, используемых Вами, для организации коммуникации с опрашиваемыми.
  • Работы по автоматической передаче сведений об опрашиваемых, необходимых для их анкетирований, из информационных ресурсов, используемых Вами в Базу данных опрашиваемых проекта.
  • Работы по синхронизации функциониования рассылочных сервисов проекта и Ваших информационных ресурсов.
  • Работы по организации трансформации массивов собранных анкет в форматы, подходящие для автоматической передачи массивов собранных анкет в информационные системы и программные средства, используемые Вами для самостоятельной обработки и анализа собранной информации.
  • Работы по организации трансформации массивов собранных Вами анкет с помощью каких-либо сторонних систем анкетирования в формат, подходящий для автоматического внедрения массивов собранных анкет в программные средства проекта, для последующей обработки и анализа собранной информации в рамках проекта.
  • Работы по настройке используемого в рамках проекта программного обеспечения и информационных систем для создания Вам возможностей необходимого контроля за прохождением опрашиваемыми анкетирований при необходимой синхронизации с программным обеспечением и иноформационными системами, используемыми Вами.
  • Работы по настройке используемого в рамках проекта программного обеспечения для создания возможностей изготовления дополнительных отчетных форм, необходимых Вам.
  • Работы по усовершенствованию настроек используемого в рамках проекта программного обеспечения в Ваших интересах.

Далее, при использовании Вашего технического содействия, административного ресурса, и, при необходимости, информационных систем осуществляется опрос, в следующих возможных вариантах:

  • Размещение гиперссылок на электронную анкету на определенном сайте или сайтах;
  • Размещение гиперссылок на электронную анкету на определенных страничках в определенных социальных сетях;
  • Размещение гиперссылок на электронную анкету в определенных группах в определенных мессенджерах;
  • Размещение гиперссылок на электронную анкету в определенных каналах в определенных мессенджерах;
  • Размещение гиперссылок на электронную анкету в личных кабинетах опрашиваемых;
  • Рассылка гиперссылок на электронную анкету опрашиваемым по электронной почте;
  • Рассылка гиперссылок на электронную анкету опрашиваемым через смс-сообщения на их телефонные номера;
  • Рассылка гиперссылок на электронную анкету опрашиваемым через сообщения в мессенджерах по их телефонным номерам или идентификаторам;
  • Рассылка гиперссылок опрашиваемым через сообщения в социальных сетях по их идентификаторам.

Возможны и иные варианты организации Интернет-анкетирования.

Возможны следующие варианты допуска к прохождению анкетирований:

  • Свободный;
  • По одному заданному паролю;
  • По группе заданных паролей;
  • По группе случайно сгенерированных паролей;
  • По индивидуальным индентификаторам и паролям;
  • По наименованию E-mail;
  • По номеру мобильного телефона.

Возможны и иные варианты организации доступа к прохождению Интернет-анкетирований.

Возможно обеспечение полной анонимности опрашиваемых при сохранении необходимого уровня контроля за процессом анкетирований.

Возможно установление ограничений на число анкет, которое может заполнить один опрашиваемый.

Возможен контроль необходимости дозаполнения опрашиваемым своей анкеты.

Возможна верификация (проверка на достоверность) ответов опрашиваемых.

Возможен контроль факта прохождения каждым конкретным опрашиваемым анкетирования.

Возможна организация непрерывного отслеживания Вами процесса анкетирования.

Приветствуется предоставление Вами опрашиваемым каких-либо льгот или скидок на что-либо, что технически совместными усилиями возможно обеспечить.

Далее, по окончании проведения исследований, со своей стороны, согласно прайс-листу:

  • Осуществляю, при необходимости, корректировку массива собранных электронных анкет в соответствии с заявленными параметрами выборочной совокупности.
  • Осуществляю построение всей необходимой отчетности (предусмотренную в Прайс-листе в разделе: “Услуги, оказываемые после проведения опросов”).
